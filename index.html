Welcome to Team Elk's website!

We are a project group from the KTH course II1305. Our project is to create a 3D multiplayer video game using unity based on the boardgame roborally. The game has several phases in which the player first programmes their robot and then moves during the execution phase. The main goal is to be the first to acquire 3 flags.
